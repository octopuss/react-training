react-training
==============

React training for GDEV team

Part 0
------

*   Gulp
*   Sass
*   Webpack

*Usage*
you need to have node.js, git installed
bower and gulp as global npm package

    npm install -g bower
    npm install -g gulp

than run commands below to initialize project

    npm install
    bower install


use

    gulp

command to process gulp tasks
